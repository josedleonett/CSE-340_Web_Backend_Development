# CSE 340: Web Backend Development

BYU-Idaho CSE 340 course repository - Web Backend Development using Node.js, Express, and PostgreSQL.

## Course Structure

- **Week 01**: Server-side Fundamentals
- **Week 02**: Database Interaction
- **Week 03**: The Model-View-Controller Pattern
- **Week 04**: Inserting and Updating Data
- **Week 05**: Authentication and Authorization
- **Week 06**: Final Feature Work
- **Week 07**: Conclusion

## Technologies

- Node.js & NPM
- Express.js
- PostgreSQL with PG Admin
- EJS Templates
- JavaScript (ES6+)

## Setup

1. Install [Node.js](https://nodejs.org/)
2. Install [PG Admin](https://www.pgadmin.org/download/)
3. Clone this repository
4. Run `npm install` to install dependencies
5. Create `.env` file with your database credentials
